1. 프로젝트 개요
PJT 명 서울·경기 지역문화 추천 서비스 (CultureRoute)
단계 10 PJT
진행일자 2026.05.29
예상 구현 시간 필수기능 6H / 심화기능 2H
개발 언어 Python 3.11 / JavaScript
프레임워크 Django 5.2 / Bootstrap 5.3 / Axios
2. 목표
• AJAX와 JSON에 대하여 이해할 수 있다.
• 비동기 통신을 이용하여 데이터를 생성, 조회, 수정, 삭제할 수 있는 애플리케이션을 제작할 수 있다.
• Many to one relationship(N:1)에 대하여 이해한다.
• Many to many relationship(N:M)에 대하여 이해한다.
• Kakao Map API를 활용하여 위치 기반 문화 장소 검색 기능을 구현할 수 있다.
• 외부 API(한국관광공사, OpenWeatherMap, Kakao)를 문서 기반으로 분석·활용할 수 있다.

6. 프로젝트 관리 (NF1001)
Branch 전략
• main — 최종 배포 브랜치
• develop — 통합 개발 브랜치
• feature/기능명 — 기능별 개발 브랜치
Commit 메시지 규칙
• feat: 새로운 기능 추가
• fix: 버그 수정
• docs: 문서 수정
• style: 코드 포맷팅
• refactor: 코드 리팩토링
